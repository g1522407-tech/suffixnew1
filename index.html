名詞の目印 (Noun Suffixes)
-ness 性質・状態を表す effectiveness, kindness
-ity 状態・程度・度合い availability, security
-tion / -sion 動作・行為・結果 production, decision
-ment 動作の結果・状態・手段 agreement, management
-ance / -ence 行為・状態・性質 assistance, difference
-ery / -ry 行為・業界・場所・全体 delivery, machinery
動詞の目印 (Verb Suffixes)
-ify / -fy 〜にする、〜化する simplify, clarify
-ate 〜する、行う evaluate, create
-en 〜にする、〜になる strengthen, broaden
-ize / -ise 〜化する、〜にする organize, realize
形容詞の目印 (Adjective Suffixes)
-ful 〜に満ちた、〜がある useful, powerful
-ive 〜の性質を持つ、〜を伴う effective, extensive
-al / -ical 〜に関する、〜の性質の professional, critical
-able / -ible 〜できる、値する reliable, flexible
副詞の目印 (Adverb Suffixes)
-ly 〜のように、〜の方法で significantly, effectively
-wise 方向、方法、〜に関して otherwise, clockwise
